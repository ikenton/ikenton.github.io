Mental Health App
Created a backend using AWS for a Mental Health App. The app was a way to keep track of the user's mental state, provide them with resources and contact medical services and friends if the app detected the user was in need of serious help.

This project was a part of my Software Engineering Class. I worked in a team of 5, as developer to set up the database, create queries, and connect the back to the frontend.
